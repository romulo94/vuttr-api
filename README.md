# VUTTR (Very Useful Tools to Remember)

---

**Api desenvolvida em NodeJS com PostgreSQL.**

A aplicação é um simples repositório para gerenciar ferramentas com seus respectivos nomes, links, descrições e tags.

## How to run

with `npm`

```console(javascript)
    npm install
    npm run dev
```

or

```console(javascript)
    yanr
    yarn dev
```

### How to test

with `npm`

```console(javascript)
    npm install
    npm run test
```

or

```console(javascript)
    yanr
    yarn test
```
